# AI Use Disclosure Draft

This project evaluates large language models as the object of study. The
generation model, orchestration prompts, Jacobian Lens extraction, and
HealthBench rubric-grading model should be reported in the Methods section with
model names, versions, manufacturers, dates of use, prompts, and analysis code.

If AI tools are used for manuscript preparation, revision, code drafting, or
formatting, the Acknowledgment section should include:

- Tool name and version.
- Manufacturer.
- Date or date range of use.
- Manuscript or code portions assisted by the tool.
- Confirmation that the authors reviewed all content and take responsibility
  for the integrity and accuracy of the final work.

References must be checked with standard reference management tools and should
not be generated or formatted solely by an AI system.
