# JAMA Network Open Package Checklist

- Original Investigation, cross-sectional simulation study.
- Structured abstract and 3 Key Points.
- Main text target: 3000 words.
- No more than 5 main tables/figures.
- Data Sharing Statement.
- STROBE checklist, adapted to public benchmark simulation.
- AI-use disclosure for research and manuscript preparation.
- Protocol/SAP as supplement.
- Supplemental prompts and model configuration.
- Supplemental scoring template and grader-decision audit trail.
- Code and public-data access statement.
